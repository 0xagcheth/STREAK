# Visual Direction

## Главная идея

Не менять текущий визуальный фундамент, а расширить его до формата `underground comic + dispatch console + visual novel`.

Сейчас у проекта уже есть сильный стиль: бумага, черная тушь, яркие акценты, постерная композиция. Для кампании нужно не изобретать второй отдельный стиль, а надстроить поверх него повествовательный слой.

## Рабочее название направления

`Underground Comic Dispatch`

## Визуальные столпы

### 1. Печатный комикс

- Бумажная фактура.
- Жесткий черный контур.
- Грязноватые, но контролируемые цветовые пятна.
- Ощущение, что интерфейс напечатан, проштампован и собран вручную.

### 2. Диспетчерская / досье

- Карточки вызовов.
- Отчеты, штампы, теги статуса, заметки маркером.
- Экран должен иногда чувствоваться как рабочая доска оператора, а не только как "игровой UI".

### 3. Visual novel-подача

- Крупные портреты персонажей.
- 2-4 состояния эмоций на ключевого героя.
- Небольшие, выразительные кадры вместо дорогой анимации.

## Как должны выглядеть экраны

### Экран смены / хаб

- Список входящих кейсов.
- Несколько стикеров, заметок и меток риска.
- Маленькие карточки персонажей, которые сейчас "на линии".

### Экран диалога

- Большой портрет слева или справа.
- Диалоговый блок снизу.
- Небольшая панель контекста: кейс, время, район, уровень риска.

### Экран матча

- Основа остается текущей.
- Добавляется сюжетная рамка: имя кейса, короткая цель, реплики персонажей, статус операции.
- Итог матча должен ощущаться как отчет по ситуации, а не просто набор очков.

### Экран последствий

- Штампованный report screen.
- Изменение отношений.
- Что открылось дальше.
- Короткая пост-сцена или сообщение.

## Палитра

Базу лучше оставить из текущей темы:

- теплый paper/cream;
- черная тушь;
- pink;
- purple;
- teal;
- orange.

Для кампании можно добавить два служебных цвета:

- warning yellow для тревоги;
- dirty navy для ночных интерфейсных слоев.

## Что рисовать в первую очередь

Не начинай с десятков иллюстраций. Сначала нужен минимальный набор:

- 3 ключевых персонажа;
- по 3 эмоции на каждого;
- 3 фоновых места;
- 1 шаблон briefing screen;
- 1 шаблон result/report screen;
- 1 набор UI-элементов: стикеры, штампы, ярлыки, скотч, маркерные пометки.

## Правило движения

Анимация должна быть дешевой в производстве, но выразительной:

- быстрые cut-in появления;
- сдвиг листов и карточек;
- stamp / flash / shake эффекты;
- минимум сложной покадровой анимации.

## Чего избегать

- Не уходить в полностью анимешную VN-эстетику.
- Не делать второй отдельный sci-fi интерфейс поверх уже найденного comic-стиля.
- Не перегружать экран декоративными слоями там, где нужна читаемость матча.

## Формула

Если коротко, визуальная формула проекта такая:

`карточный инди-комикс + ночная диспетчерская + бумажное досье + минималистичная visual novel подача`
