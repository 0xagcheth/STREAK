# STREAK Project Hub

Главная цель: превратить STREAK из чисто матчевой карточной игры в стильную сюжетную кампанию, где каждая партия ощущается как напряженная смена в странном городе.

## One-line pitch

STREAK — это push-your-luck игра с механической базой в духе Flip 7, но поданная как ночная диспетчерская драма: игрок принимает вызовы, ведет рискованные кейсы и решает, когда продолжать давить удачу, а когда фиксировать результат.

## Что уже есть

- Рабочая карточная основа и темп коротких напряженных раундов.
- Сильный визуальный фундамент: underground comic / editorial comic.
- Подходящий тон для сюжетной надстройки: взрослый, немного нервный, городской.

## Во что это превращаем

- Между матчами: диалоги, звонки, переписка, досье, решения.
- Во время матча: не абстрактный раунд, а конкретный кейс с растущим давлением.
- После матча: последствия, отношения, новые ветки, новые задания.

## Главный принцип

Сначала делаем один вертикальный срез:

Пролог -> 1 смена -> 1 кульминационный матч -> 1 экран последствий

Если этот кусок работает, только тогда наращиваем главы, персонажей и развилки.

## Решение по формату кампании

Лучший вариант для масштаба этого проекта: не полноценная визуальная новелла на десятки часов, а case-based структура в духе dispatch-игры.

Это значит:

- есть центральная смена / хаб;
- есть входящие кейсы;
- каждый кейс приводит в карточный раунд;
- после кейса мир и персонажи немного сдвигаются.

Так ты получаешь сюжет, атмосферу и персонажей, не взрывая объем производства.
